# Happy Birthday Chaile 🌸 — Upgraded Website

## What's new in this version

### 🪐 49 Photo Planets across 5 Orbital Rings
Drop your 49 photos into the `images/` folder named:
  chaile1.jpg, chaile2.jpg, ... chaile49.jpg

Ring breakdown:
  💗 Ring 1 (Pink glow)   — 8 planets, closest orbit
  ✨ Ring 2 (Gold glow)   — 10 planets
  💙 Ring 3 (Blue glow)   — 11 planets
  💜 Ring 4 (Purple glow) — 10 planets
  🩵 Ring 5 (Teal glow)   — 10 planets, outermost orbit

### New Fun Features
  🎉 Confetti burst when greeting screen appears
  ✨ Sparkle emoji trail (stars, flowers, butterflies) follows your mouse
  🌠 Shooting stars streak through the galaxy every ~3 seconds
  🌟 Thousands of background twinkle stars fill the universe
  🎊 Visit counter — tracks how many of her 49 planets you've explored
  💗 Each ring has a unique neon glow color (pink/gold/blue/purple/teal)

## Setup
1. Create an `images/` folder next to index.html
2. Add your 49 photos as chaile1.jpg ... chaile49.jpg
3. Open index.html in a browser (or serve locally with: npx serve .)

## File structure
  index.html   — main page
  style.css    — all styles & animations
  script.js    — Three.js galaxy + all features
  images/      — add your 49 photos here
